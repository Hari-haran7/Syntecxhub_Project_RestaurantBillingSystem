# Restaurant Billing System

A console-based Restaurant Billing System developed in Java.

## Features

- Add food items to the bill
- Remove the last added item
- Generate itemized receipt
- Automatic GST (18%) calculation
- Menu-driven interface
- Uses Object-Oriented Programming concepts

## Technologies Used

- Java
- ArrayList
- Scanner Class

## Menu Options

1. Burger
2. Pizza
3. Coke
4. Remove Last Item
5. Generate Bill
6. Exit

## How to Run

Compile:

javac *.java

Run:

java Main
